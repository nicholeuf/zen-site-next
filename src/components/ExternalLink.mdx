# ExternalLink

A small wrapper that renders an anchor (`<a>`) using MUI `Box` so theme-aware
styles and the `sx` prop are supported. This is intended for links that
navigate to external sites and therefore defaults to `target="_blank"` and
`rel="noopener"`.

The component accepts the normal anchor props (href, target, rel) plus the
optional `sx` and `className` props for styling.
